Verification of Parallel and Concurrent Software

… a static formal verifcation tool that provides you program correctness under different concurrency models, written in high-level programming languages, e.g. in Java, OpenCL and OpenMP; and reason not only about race freedom and memory safety, but also about functional correctness!
Released for free under the License Formal Methods and Tools, University of Twente
